# timelapse-maker

Python script to create timelapses from image sequences.

## Setup

1. Clone repository
2. Install dependencies (in virtual env):
   ```
   pip install -r requirements.txt
   ```

## Usage

- everything you'll need in inside `v1.py`

## Requirements

- Python 3.8+
- OpenCV
- NumPy

See `requirements.txt` for versions.
